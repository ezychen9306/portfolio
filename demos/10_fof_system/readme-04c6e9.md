# 智能投顾系统（FOF）

## 项目概述

星辰资本智能投顾系统，基于用户风险偏好画像进行 FOF（Fund of Funds）产品推荐。

## 核心功能

### 1. 风险偏好评估
- 问卷调查
- 历史投资行为分析
- 风险承受能力测评

### 2. 资产配置建议
- 股票/债券/货币基金配比
- 风险分散策略
- 动态再平衡

### 3. 产品推荐
- 基金筛选与评级
- 个性化推荐
- 组合优化

### 4. 投资监控
- 收益追踪
- 风险预警
- 定期报告

## 业务成果

- 服务用户 **5 万+**
- 管理资产规模 **8 亿+**
- 平均年化收益 **12.5%**
- 用户满意度 **4.3/5**

## 技术架构

- Python（量化模型）
- 多因子模型（宏观/财报/量价/舆情）
- 机器学习（推荐算法）
- Web 前端

## 示例配置（脱敏）

### 保守型投资者
- 股票型基金：20%
- 债券型基金：60%
- 货币基金：20%
- 预期年化收益：6-8%

### 激进型投资者
- 股票型基金：70%
- 债券型基金：20%
- 货币基金：10%
- 预期年化收益：15-20%
