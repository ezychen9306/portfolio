# Portfolio 项目示例库

本目录存放所有用于 Portfolio 展示的**脱敏示例项目**。

## 重要说明

⚠️ **所有项目均为脱敏版本，仅用于展示，不包含真实业务数据**

- 数据已完全脱敏或使用模拟数据
- 不影响真实项目文件
- 每个项目独立存放，便于维护

## 核心项目（14 个）

### 风险策略类（6 个）
1. `01_disaster_management/` - 灾难管控策略
2. `02_quota_strategy_v5/` - 额度策略 V5
3. `03_rm_system/` - RM 风险管理系统
4. `04_ai_deep_strategy/` - AI 风控策略
5. `05_kyc_platform/` - KYC 策略中台
6. `06_telesales_system/` - 智能电销系统

### 数据分析与报告类（2 个）
7. `07_year_end_report/` - 25年年度总结（链至实际报告）
8. `08_weekly_monitoring/` - 目标达成体系

### AI 与投资类（2 个）
9. `09_ai_sharing/` - AI 内部分享
10. `10_fof_system/` - 智能投顾系统（FOF）

### 补充项目（4 个）
13. `13_knowledge_base_agent/` - 知识库 Agent
14. `14_skills_development/` - Skills 技能开发

## 使用说明

1. 每个项目文件夹包含：
   - `demo.html` - 可交互的演示页面（部分项目）
   - `readme.md` - 项目说明
   - 其他必要的示例文件

2. 生成项目配图：
   - 使用 canvas-design skill 生成配图
   - 保存到 `../assets/images/projects/`

3. 添加新项目：
   - 创建新文件夹
   - 添加脱敏示例文件
   - 更新本 README
