# RM 风险管理系统

## 项目概述

全业务风险管理（Risk Management）系统，对 7 条差异显著业务线进行终损预估，支撑部门经营、公司财报及外部审计。

## 核心功能

### 1. 多业务线风险预估
- 7 条业务线独立建模
- Cohort 分析与 Vintage 曲线
- 终损率预测（ECL 模型）

### 2. 实时监控与预警
- 日度风险指标更新
- 异常波动自动预警
- 多维度下钻分析

### 3. 报表自动化
- 月度经营报告
- 季度 QBR 报告
- 年度审计报告

## 业务价值

- 支撑 **500 亿+** 在贷余额管理
- 为财报提供准确的拨备计提依据
- 满足外部审计合规要求
- 辅助经营决策与资源配置

## 技术架构

- Python（建模与预测）
- SQL/ODPS（数据处理）
- Airflow（调度）
- HTML 报表（可视化）
