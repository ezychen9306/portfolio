# KYC 策略中台

## 项目概述

平安普惠 0-1 搭建的数据与算法驱动策略中台，指导电销名单分配与利润最大化。

## 核心功能

### 1. 客户价值评估
- LTV（生命周期价值）预测
- 转化率预测模型
- 风险评分模型

### 2. 名单智能分配
- BD/电销资源优化配置
- 动态优先级排序
- 实时名单更新

### 3. 策略效果监控
- 转化率实时追踪
- 放款规模监控
- ROI 分析

## 业务成果

- 三年内转化率月均 **+15.6%**
- 月新增放款约 **2.7 亿**
- fpd7 **降 18.3%**
- 系统覆盖率 **100%**

## 技术架构

- Python（建模）
- SQL（数据处理）
- Redis（实时缓存）
- Web 管理后台
