# 智能电销系统

## 项目概述

平安普惠带领 4 人从零开发，算法为 BD/电销座席动态推荐策略及话术。

## 核心功能

### 1. 实时策略推荐
- 客户画像实时分析
- 最优产品推荐
- 话术模板生成

### 2. 动态话术优化
- 基于历史成功案例
- A/B 测试优化
- 个性化话术调整

### 3. 座席辅助
- 实时风险提示
- 异议处理建议
- 成交概率预测

## 业务成果

- 月均转化率 **+9.8%**
- 月新增放款 **1.1 亿**
- 系统覆盖率 **100%**
- 座席满意度 **4.5/5**

## 技术架构

- Python（推荐算法）
- NLP（话术生成）
- WebSocket（实时推送）
- React（前端界面）
