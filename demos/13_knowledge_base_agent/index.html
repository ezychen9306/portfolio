知识库 Agent

项目概述

基于 RAG（Retrieval-Augmented Generation）的知识库 Agent，支持文档解析、智能检索与结构化输出。

核心功能

1. 文档解析

  • 支持格式：PDF、Word、Markdown、HTML
  • 文本提取与清洗
  • 结构化信息抽取

2. 向量化与索引

  • 文本分块（Chunking）
  • Embedding 生成
  • 向量数据库存储（Chroma/FAISS）

3. 智能检索

  • 语义相似度搜索
  • 混合检索（向量 + 关键词）
  • 重排序（Reranking）

4. 结构化输出

  • 基于检索结果生成回答
  • 引用来源标注
  • 多轮对话支持

应用场景

  • 内部文档问答
  • 政策法规查询
  • 技术文档助手
  • 客服知识库

技术架构

  • LangChain（框架）
  • OpenAI / Claude（LLM）
  • Chroma（向量数据库）
  • FastAPI（后端）

示例数据（脱敏）

  • 文档数量：500+ 篇
  • 知识条目：10,000+ 条
  • 检索准确率：**92.5%**
  • 平均响应时间：**1.2 秒**

示例对话

**用户**：菲律宾现金贷的最高额度是多少？

**Agent**：根据《菲律宾业务额度策略 V5》文档，菲律宾现金贷的最高额度为 50,000 PHP，针对优质客户可临时提额至 70,000 PHP。[来源：额度策略文档 p.12]
