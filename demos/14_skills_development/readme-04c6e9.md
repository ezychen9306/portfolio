# Skills 技能开发

## 项目概述

基于 Claude Code / Cursor 的 Skills 技能开发，将业务经验与开发规范封装为可复用的 AI 技能。

## 核心技能

### 1. constitutional-compliance
- 23 条宪法规则快速参考
- 项目规范与开发标准
- 最佳实践指南

### 2. data-report-builder
- 数据驱动 HTML 报告生成
- 交互式图表与模块化架构
- 自动化报表流水线

### 3. year-end-report
- 年终汇报幻灯片生成
- 策略体系图与时间线
- 月报/QBR 链接整合

### 4. wechat-article-processor
- 微信文章处理与格式化
- 内容提取与结构化
- 批量处理自动化

## 技能特点

- **可复用**：一次开发，多次使用
- **标准化**：统一规范，减少错误
- **高效**：自动化流程，节省时间
- **可维护**：版本管理，持续优化

## 开发流程

1. 需求分析与场景梳理
2. 技能设计与接口定义
3. 代码实现与测试
4. 文档编写与示例
5. 部署与迭代优化

## 业务价值

- 开发效率提升 **50%+**
- 代码质量提升 **30%+**
- 规范遵守率 **95%+**
- 团队协作效率 **40%+**
