# 周报监控

## 项目概述

自动化生成周度风险监控报告，包含核心指标、异常预警、策略效果追踪。

## 核心功能

### 1. 核心指标监控
- 放款规模与结构
- 逾期率（fpd7/fpd30/M3+）
- 在贷余额与增长
- 客群分布变化

### 2. 异常预警
- 指标异常波动检测
- 策略效果异常提示
- 数据质量问题预警

### 3. 策略效果追踪
- 新上线策略效果
- A/B 测试进展
- 策略迭代建议

## 报告特点

- **自动化**：每周一自动生成
- **可视化**：交互式图表
- **可追溯**：历史数据对比
- **可分享**：HTML 格式，易传阅

## 技术栈

- Python（数据处理）
- SQL/ODPS（数据提取）
- Chart.js（可视化）
- HTML（报告生成）
