# 客群分层策略体系

## 项目概述

乐信普惠贷中风险策略，通过客群分层实现差异化策略匹配与版本化管理。

## 核心功能

### 1. 客群分层模型
- 基于信用评分、行为特征、资产状况
- 5 层客群划分（优质/标准/成长/观察/风险）
- 动态调整机制

### 2. 差异化策略
- 额度策略：不同客群不同额度
- 定价策略：风险定价差异化
- 审批策略：审批通过率差异化

### 3. 版本化管理
- 策略版本控制
- A/B 测试验证
- 灰度发布机制

### 4. 小微线下策略
- 白名单机制
- 线下场景差异化
- 风险与规模平衡

## 业务成果

- 年化风险压降 **11.6%**
- 放款规模 **+15.3%**
- 盈利性 **+128.8%**
- 小微白名单上线次月：风险降 **23%**，规模 **+16%**

## 技术栈

- Python（建模）
- SQL（客群标签）
- 策略引擎
- 监控看板
