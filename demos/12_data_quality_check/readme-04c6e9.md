# 数据质量校验工具

## 项目概述

自动化数据质量校验工具，检测数据形状、缺失值、异常值、一致性等问题。

## 核心功能

### 1. 形状校验
- 行数/列数检查
- 数据类型验证
- 字段完整性检查

### 2. 缺失值检测
- 缺失率统计
- 缺失模式分析
- 缺失值填充建议

### 3. 异常值检测
- 统计方法（3σ、IQR）
- 业务规则校验
- 异常值标记与处理

### 4. 一致性校验
- 跨表关联校验
- 逻辑一致性检查
- 时间序列连续性

### 5. 自动化报告
- HTML 可视化报告
- 问题汇总与建议
- 历史趋势对比

## 业务价值

- 数据问题发现率 **95%+**
- 人工校验时间节省 **80%+**
- 数据质量提升 **30%+**

## 技术栈

- Python（pandas, numpy）
- Great Expectations
- HTML 报告生成
- 自动化调度

## 示例校验（脱敏）

### 数据集：客户交易表
- 总行数：1,250,000
- 总列数：45
- 缺失值：3 个字段，缺失率 <1%
- 异常值：金额字段 125 条异常（0.01%）
- 一致性：客户 ID 关联校验通过
- 结论：✓ 数据质量良好
