# 🌐 GitHub Pages 设置指南

## ⚠️ 重要：需要进入仓库的 Pages 设置

你刚才看到的是**个人 Profile 的 Pages 设置**（Verified domains），不是仓库的 Pages 设置。

## ✅ 正确的步骤

### 方法 1: 直接访问仓库 Pages 设置

访问这个链接：
**https://github.com/ezychen9306/portfolio/settings/pages**

### 方法 2: 从仓库页面进入

1. **进入你的仓库**
   - 访问：https://github.com/ezychen9306/portfolio
   - 或点击 "Go to your personal profile" 返回，然后找到 `portfolio` 仓库

2. **打开 Settings**
   - 在仓库页面顶部，点击 **"Settings"** 标签

3. **找到 Pages**
   - 在左侧菜单中，向下滚动
   - 找到 **"Pages"** 选项（在 "Security" 和 "Secrets and variables" 附近）
   - 点击 **"Pages"**

4. **配置 Pages**
   - 在 "Source" 部分：
     - 选择 `Deploy from a branch`
     - Branch: 选择 `main`
     - Folder: 选择 `/ (root)`
   - 点击 **"Save"** 按钮

5. **等待部署**
   - 页面会显示：`Your site is live at https://ezychen9306.github.io/portfolio/`
   - 等待 1-2 分钟让 GitHub 完成构建

## 🌐 你的网站地址

配置完成后，网站地址将是：

**https://ezychen9306.github.io/portfolio/**

## 📋 区别说明

- ❌ **个人 Profile Pages** (`github.com/settings/pages`)：用于个人主页，需要验证域名
- ✅ **仓库 Pages** (`github.com/用户名/仓库名/settings/pages`)：用于项目网站，直接部署代码

你需要的是**仓库 Pages**，不是个人 Profile Pages。
