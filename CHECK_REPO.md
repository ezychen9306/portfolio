## 如果仓库已创建但推送失败

可能需要认证，请：
1. 访问：https://github.com/settings/tokens
2. Generate new token (classic)
3. 勾选 `repo` 权限
4. 生成并复制 token
5. 推送时：
   - Username: 你的 GitHub 用户名
   - Password: 粘贴 token
